RoboVisionEV3

RoboVisionEV3 is an app and EV3 Robot with Remote control App over bluetooth and Live Wifi Camera using small smartphone and IP WEBCAM app to view live video via Web Browser.
Details

I built an Robot with Lego Mindstorms EV3 with 2 Large Motors and one small. I built holder for phone and powerbank using Legos. Then I put the phone holder with Smartphone to Small Motor. Then I Programmed app with Remote control without Lego Mindstorms Developing. I used Bluetooth to control Robot and Web Browser to see the Live Video in app.
